What is an FPS game?

an fps game is any video game that features some sort of weapon and a first-person perspective. usually first person shooter games have 3d environments with player controlled movement, but this isn't a hard rule. within these limits, they can be almost anything. some of the subgenres include tactical shooters, hero shooters, mil sims, extraction shooters, and battle royales. uwu pat pat
